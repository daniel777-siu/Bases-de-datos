
Proyecto: Sistema de Gestión Académica - Universidad Virtual

Este proyecto contiene el script SQL completo que incluye:
- Creación de base de datos
- Tablas relacionales con claves primarias y foráneas
- Inserción de datos de prueba
- Vista para visualizar inscripciones
- Creación de usuarios y permisos

Archivo incluido:
- universidad_gestion.sql : Script completo para importar en MySQL

Instrucciones:
1. Abre tu gestor de base de datos (como MySQL Workbench o DBeaver)
2. Ejecuta el archivo universidad_gestion.sql
3. Verifica que se hayan creado correctamente las tablas y la vista
4. Opcional: Crea capturas de pantalla mostrando la vista y los permisos
